# FourCutPhoto

FourCutPhoto는 PyQt6를 사용하여 만든 네 컷 사진 촬영 애플리케이션입니다. 사용자는 이 프로그램을 통해 손쉽게 네 컷 사진을 촬영하고, 원하는 프레임을 적용하여 이미지를 생성할 수 있습니다.

## 주요 기능

-   **환영 화면**: 애플리케이션 시작 시 사용자 환영
-   **프레임 선택**: 다양한 디자인의 프레임 중 하나를 선택
-   **사진 촬영**: 웹캠을 이용하여 실시간으로 사진 촬영
-   **사진 확인 및 재촬영**: 촬영된 사진을 확인하고, 마음에 들지 않으면 재촬영 가능
-   **이미지 생성**: 선택된 사진과 프레임을 합성하여 최종 네 컷 이미지 생성

## 애플리케이션 흐름

1.  **시작**: 환영 화면에서 시작합니다.
2.  **프레임 선택**: 제공되는 여러 프레임 디자인 중 하나를 선택합니다.
3.  **사진 촬영**: 웹캠이 활성화되고, 사용자는 총 4장의 사진을 촬영합니다.
5.  **사진 확인**: 촬영된 4장의 사진을 확인합니다. 이 단계에서 재촬영을 선택할 수 있습니다.
6.  **편집**: 선택된 사진들이 프레임과 합쳐져 최종 이미지가 생성됩니다.
7.  **종료 및 재시작**: 모든 과정이 끝나면 다시 초기 프레임 선택 화면으로 돌아가 새로운 네 컷 사진을 만들 수 있습니다.

## 주요 의존성

-   `pyqt6`
-   `opencv-python`
-   `pillow`
-   `numpy`

## 실행 방법

1.  저장소를 복제하거나 다운로드합니다.

2.  프로젝트 루트 디렉토리에서 다음 명령어를 실행하여 필요한 라이브러리를 설치합니다.

    ```bash
    pip install numpy opencv-python pillow pyqt6
    ```

3.  다음 명령어를 사용하여 애플리케이션을 실행합니다.

    ```bash
    python src/main/qt_main.py
    ```
